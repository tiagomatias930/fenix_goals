# 🦅 Fênix Goals

**Fênix Goals** é uma plataforma de alta performance desenvolvida para transformar objetivos abstratos em conquistas tangíveis. O sistema é fundamentado nos **12 Passos do Seminário Fênix de Brian Tracy**, combinando psicologia de sucesso com inteligência artificial generativa.

## 🧠 A Metodologia
Diferente de simples listas de tarefas, o Fênix Goals força o usuário a atravessar o processo de cristalização mental:

1.  **Desejo Ardente:** Quantificação da intensidade do desejo (1-10).
2.  **Crença e Convicção:** Alinhamento de valores.
3.  **Cristalização (Escrita):** Confirmação do compromisso físico.
4.  **Análise de Motivação:** Definição dos "Porquês" (o combustível).
5.  **Ponto de Partida:** Honestidade brutal sobre a situação atual.
6.  **Prazos:** Definição de datas limites claras.
7.  **Teoria das Restrições:** Identificação do obstáculo principal (gargalo).
8.  **Conhecimento Adicional:** Habilidades necessárias para o próximo nível.
9.  **Alianças Estratégicas:** Identificação de pessoas e grupos de apoio.
10. **Plano de Ação:** Decomposição em tarefas granulares.
11. **Visualização (AI):** Geração de imagens motivacionais via Gemini API.
12. **Persistência Inabalável:** Monitoramento contínuo via Dashboard.

## 🚀 Funcionalidades Principais

-   **Fluxo Wizard Dirigido:** Um guia passo a passo que impede a criação de metas rasas.
-   **Dashboard de Performance:** Visualização de progresso com gráficos de linha do tempo SVG dinâmicos.
-   **IA Visualizer:** Integração com a `gemini-2.5-flash-image` para gerar imagens inspiradoras baseadas na descrição da meta.
-   **Gerador de Certificados:** Emissão de certificados de conquista (formato para impressão) ao concluir um objetivo.
-   **Countdown Inteligente:** Rastreador de tempo real para o deadline da meta.
-   **Interface Dark Premium:** Design focado em concentração e seriedade, utilizando Tailwind CSS.

## 🛠️ Stack Técnica

-   **Frontend:** React 19 (Hooks, Context, Memoization).
-   **Estilização:** Tailwind CSS (Custom Theme).
-   **Ícones:** Lucide React.
-   **IA:** Google Generative AI SDK (@google/genai).
-   **Arquitetura:** Componentização baseada em estados claros e tipos TypeScript rigorosos.

## 📦 Como Utilizar

1.  **Defina sua Meta:** Use o botão "Novo Objetivo" para iniciar o Wizard.
2.  **Seja Específico:** A IA e os indicadores de progresso funcionam melhor com dados precisos.
3.  **Visualize diariamente:** Utilize o Dashboard para marcar o progresso do Plano de Ação.
4.  **Conquiste:** Ao atingir 100%, gere seu certificado de excelência.
